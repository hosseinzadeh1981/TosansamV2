🎯 خلاصه اجرایی

پروژه TosansamV2 یک Code Generator پیشرفته برای اکوسیستم .NET Core است که با معماری Clean طراحی شده. در فاز اول، Generator مدل‌های MVC با موفقیت پیاده‌سازی شده و آماده توسعه بیشتر است.

🏗️ معماری سیستم

// ساختار Clean Architecture Presentation Layer (CLI) ← Tosansam.Cli Business Logic Layer ← Tosansam.Generators Core Layer ← Tosansam.Core (Entities + Interfaces)

✅ دستاوردهای فاز ۱

  • ✅ Generator MVC Core - تولید کد C# معتبر
  • ✅ ساختار Clean Architecture - پیاده‌سازی استاندارد
  • ✅ ذخیره خودکار فایل (SaveToFile) - با encoding UTF-8
  • ✅ مدیریت Git حرفه‌ای - کامیت‌های معنادار
  • ✅ مستندات کامل - README + گزارش داخلی

🔧 نمونه کد عملی

// تولید مدل Customer var generator = new ModelGenerator(); var table = new TableDefinition { ModelName = "Customer", Fields = new List<FieldDefinition> { new() { FieldName = "Id", DataType = "int" }, new() { FieldName = "Name", DataType = "string" }, new() { FieldName = "Email", DataType = "string" }, new() { FieldName = "Age", DataType = "int" } } }; string code = generator.Generate(table); // ذخیره فایل var fileService = new FileService(); fileService.SaveModel(table, code); // ✅ خروجی: GeneratedModels/Customer.cs
