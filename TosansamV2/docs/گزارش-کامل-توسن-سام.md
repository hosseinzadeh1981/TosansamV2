## 🎯 خلاصه اجرایی:
پروژه TosansamV2 یک Code Generator پیشرفته برای اکوسیستم .NET Core است که با معماری Clean طراحی شده. 
در فاز اول، Generator مدل‌های MVC با موفقیت پیاده‌سازی شده و آماده توسعه بیشتر است.

## 📊 نمونه کد عملی:
\\\csharp
// تولید مدل Customer
var generator = new ModelGenerator();
var table = new TableDefinition 
{
    ModelName = \"Customer\",
    Fields = new List<FieldDefinition>
    {
        new() { FieldName = \"Id\", DataType = \"int\" },
        new() { FieldName = \"Name\", DataType = \"string\" }
    }
};

string code = generator.Generate(table);
